# PDF Q&A RAG Pipeline

PDF Q&A system using RAG pipeline with ChromaDB vector search and local LLMs (Gemma3:1b, Qwen2.5, Llama2). Perplexity-style citations with page-level source tracking.

## Dataset
**Download from Kaggle:** [Dataset of PDF Files](https://www.kaggle.com/datasets/manisha717/dataset-of-pdf-files)

> **About Dataset:** Diverse PDF files covering science, technology, history, literature, business (reports, articles, manuals). 1076 PDFs, 13K+ pages.
